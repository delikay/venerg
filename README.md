# Venergi Website

Marketing website for Venergi, built with React + Vite + Tailwind CSS.

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

## Contact form configuration

The contact form submits directly to FormSubmit using a hardcoded recipient email:

```text
https://formsubmit.co/ajax/venergielectric@gmail.com
```
